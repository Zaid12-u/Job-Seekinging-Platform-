#Job Seekinging Platform 
A full-stack MERN-based job portal that connects job seekers with employers through an intuitive and secure 
platform. The system features user authentication with JWT, allowing candidates to create profiles, search for jobs, 
and submit applications seamlessly. Employers can post job listings and manage applications efficiently. Integrated 
Cloudinary for resume and profile image uploads, Redux for efficient state management across components, and 
RESTful APIs for real-time data synchronization. The platform provides a responsive user interface with dynamic 
updates, ensuring smooth navigation and an enhanced user experience for both job seekers and recruiters.
